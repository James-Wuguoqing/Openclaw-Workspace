# OpenClaw Workspace

一个面向个人使用的 OpenClaw 助手工作区。

这个仓库包含：
- 助手身份与行为规则
- 用户偏好与上下文
- memory / continuity 机制
- skills 扩展
- 一个已跑通的 Notion 财经早报自动化流程

---

## 先看哪里

建议阅读顺序：

1. `AGENTS.md`
2. `PROJECT_HISTORY.md`
3. `SOUL.md`
4. `USER.md`
5. `notion-direct/`

如果只想快速看懂仓库，读这三个就够：
- `AGENTS.md`
- `PROJECT_HISTORY.md`
- `notion-direct/`

---

## 核心目录

- `AGENTS.md`：工作区规则与运行方式
- `PROJECT_HISTORY.md`：项目演进的叙事版说明
- `CHANGELOG.md`：版本 / 变更记录版说明
- `SOUL.md`：助手性格与行为风格
- `USER.md`：用户信息与偏好
- `memory/`：每日记忆与运行记录
- `notion-direct/`：Notion 财经早报自动化流程
- `skills/`：本地技能扩展
